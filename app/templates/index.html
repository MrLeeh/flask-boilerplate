Flask Boilerplate

This is my personal Flask-Boilerplate. It comes with support for Bootstrap, WTF and SQLAlchemy. The purpose is to minimize the effort to set up a new Website with Flask.

Features

  • Blueprint integration
  • Bootstrap integration
  • SQLAlchemy integration
  • Docker ready
  • different configuration options for production and development

Module

  • Flask
  • Flask-Bootstrap
  • Flask-Migrate
  • Flask-Script
  • Flask-SQLAlchemy
  • Flask-WTF
  • gunicorn
  • Jinja2
  • SQLAlchemy
  • WTForms

Static files served via nginx
